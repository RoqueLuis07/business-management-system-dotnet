# Business Management System

## 🧾 Description
Business management system designed for small and medium-sized businesses.
It allows managing clients, products and sales using a clean and scalable architecture.

## 🚀 Technologies
- C# (.NET)
- PostgreSQL / MySQL
- REST API
- Entity Framework Core

## 🔑 Features
- User authentication with roles
- Client management
- Product management
- Sales records
- Data validation and error handling

## 🧱 Architecture
Layered architecture separating business logic, infrastructure and presentation.
