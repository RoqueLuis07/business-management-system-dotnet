# Flujo de Órdenes de Trabajo (OT)

## Estados propuestos
1. Ingresada
2. Asignada
3. En diagnóstico
4. Esperando aprobación (si requiere presupuesto)
5. Aprobada
6. En reparación / mantenimiento
7. Terminada
8. Lista para entrega
9. Entregada / Cerrada

## Notas
- Una OT puede requerir presupuesto previo o pasar directo a reparación.
- El tablero principal del taller se basa en estos estados.
