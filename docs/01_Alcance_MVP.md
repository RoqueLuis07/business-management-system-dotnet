# Alcance MVP – Sistema de Taller y Venta de Maquinaria

## Objetivo
Digitalizar el flujo del taller: ingreso de equipos, orden de trabajo, diagnóstico, presupuesto, reparación/mantenimiento, entrega y garantía.

## Módulos incluidos (MVP)
- Clientes
- Equipos / Maquinarias
- Órdenes de Trabajo (OT)
- Diagnóstico / Ficha técnica
- Presupuesto (opcional) + Aprobación
- Repuestos / Materiales usados (simple)
- Cierre de OT + Entrega
- Garantías + Historial técnico por equipo

## Fuera de alcance (por ahora)
- Facturación electrónica / timbrado / integración contable completa
- Pagos online
- Multi-sucursal
- Roles avanzados y permisos granulares
