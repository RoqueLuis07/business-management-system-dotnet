# Reglas de Negocio

## Garantía
- La garantía comienza desde la FECHA DE ENTREGA del equipo, no desde el pago.
- Aplica incluso si el trabajo se entrega a crédito.
- Duración estándar: 30 días (configurable).

## Presupuesto
- Algunas OTs requieren presupuesto previo.
- Si está "Esperando aprobación", no se inicia la reparación hasta aprobar.

## Identificación de equipos
- Cada equipo se identifica con: tipo, marca, modelo y número de serie/motor.
- Se registran accesorios recibidos al ingreso (ej.: espada, cadena, protector, etc.).
